Elevator Controller Testing

Black box test plan for the Elevator Controller module.

Module Inputs

  • CAN Message 'SC_Cmd_To_EC' (0x100)
    • Signal 'SC_Floor_Cmd' (Bits 0-1)
    • Signal 'SC_Enable' (Bit 2)

Module Outputs

  • CAN Message 'EC_Status' (0x101)
    • Signal 'EC_Car_Pos' (Bit 0-1)

Car Controller Testing

Black box test plan for the Car Controller module.

Module Inputs

  • CAN Message 'EC_Status' (0x101)
    • Signal 'EC_Car_Pos' (Bits 0-1)
    • Signal 'EC_Status' (Bit 2)
  • CAN Message 'SC_Cmd_To_EC' (0x100)
    • Signal 'SC_Floor_Cmd' (Bits 0-1)
    • Signal 'SC_Enable' (Bit 2)
  • Door Open Button
  • Door Close Button
  • Floor 1 Button
  • Floor 2 Button
  • Floor 3 Button

Module Outputs

  • CAN Message 'CC_Status' (0x200)
    • Signal 'CC_Floor_Req' (Bit 0-1)
    • Signal 'CC_Door_State' (Bit 2)
  • Door Open LED
  • Door Closed LED
  • Floor 1 LED
  • Floor 2 LED
  • Floor 3 LED

Supervisory Controller Testing

Black box test plan for the Supervisory Controller module.

Module Inputs

  • CAN Message 'EC_Status' (0x101)
    • Signal 'EC_Car_Pos' (Bits 0-1)
    • Signal 'EC_Status' (Bit 2)
  • CAN Message 'CC_Status' (0x200)
    • Signal 'CC_Floor_Req' (Bit 0-1)
    • Signal 'CC_Door_State' (Bit 2)
  • CAN Message 'F1_Status' (0x201)
    • Signal 'F1_Call_Req' (Bit 0)
  • CAN Message 'F2_Status' (0x202)
    • Signal 'F2_Call_Req' (Bit 0)
  • CAN Message 'F3_Status' (0x203)
    • Signal 'F3_Call_Req' (Bit 0)
  • Remote Floor Request

Module Outputs

  • CAN Message 'SC_Cmd_To_EC' (0x100)
    • Signal 'SC_Floor_Cmd' (Bits 0-1)
    • Signal 'SC_Enable' (Bit 2)

Test Plans and Results

Supervisory Controller Test 1

Test Description: Confirm SC transitions from "CAR_NOT_MOVING" to "REQUEST_NEW_FLOOR" when door is closed and a floor is requested. Repeat test for each source of a floor request.

Test Setup: SC State = CAR_NOT_MOVING

Inputs:
  • EC_Car_Pos = 0
  • SM_Floor_Req = 3
  • Door_State = CLOSED
Expected Outputs:
  • SC_Floor_Cmd = 3
  • SC_Enable = 1
Test Result: Passed

Supervisory Controller Test 2

Test Description: Confirm SC remains in state "CAR_NOT_MOVING" when door is open and a floor is requested.

Test Setup: SC State = CAR_NOT_MOVING.

Inputs:
  • EC_Car_Pos = 0
  • SM_Floor_Req = 3
  • Door_State = OPEN
Expected Outputs:
  • SC_Floor_Cmd = 0
  • SC_Enable = 0
Test Result: Passed

Supervisory Controller Test 3

Test Description: Confirm SC transitions from "CAR_MOVING" to "CAR_NOT_MOVING" when elevator car reaches requested floor and door is opened.

Test Setup: SC State = CAR_MOVING

Inputs:
  • EC_Car_Pos = 0 --> 1
  • CC_Door_State = CLOSED --> OPEN
Expected Outputs:
  • SC_Floor_Cmd = 1 --> 0
  • SC_Enable = 1 --> 0
Test Result: Passed
