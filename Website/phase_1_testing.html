Elevator Controller Testing

Black box test plan for the Elevator Controller module.

Module Inputs

  • CAN Message 'SC_Cmd_To_EC' (0x100)
    • Signal 'SC_Floor_Cmd' (Bits 0-1)
    • Signal 'SC_Enable' (Bit 2)

Module Outputs

  • CAN Message 'EC_Status' (0x101)
    • Signal 'EC_Car_Pos' (Bit 0-1)
    • Signal 'EC_Status' (Bit 2)

Test Plans and Results

Position Test 0

Test Description: Confirm EC reports "Floor 0" when car is in between floors.

Test Setup: Move elevator car to 100cm height marker.

Inputs:
  • None
Expected Outputs:
  • EC_Car_Pos = 0
Test Result: Passed
